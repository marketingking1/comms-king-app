# Ganchos Visuais e de Edição — Referência Completa

Todos os ganchos visuais, técnicas de edição e estratégias visuais para criativos em vídeo.

> "Gancho é VERBAL + VISUAL juntos. Copy boa + edição chata = Hook rate baixo. Copy média + edição dinâmica = Hook rate alto."

---

## GANCHO VISUAL 1: CORTES RÁPIDOS

**Descrição:** Dinâmica frenética de edição que impede o cérebro de processar antes de prender.

**Técnica:**
- Mudar cena a cada 1-1,5 segundos
- Mínimo 3 cortes nos primeiros 5 segundos
- Ritmo frenético antes do cérebro processar

**Quando usar:**
- Produtos físicos (e-commerce)
- Lifestyle/rotina
- Demonstrações rápidas
- Antes/depois físico

**Estrutura temporal:**
```
[0-1,5s]: Produto na mão
[1,5-3s]: Pessoa usando
[3-4,5s]: Resultado/benefício
[4,5s+]: Narração começa
```

**Adaptação King:**
```
[0-1,5s]: Tela do app/plataforma
[1,5-3s]: Aluno falando em inglês
[3-4,5s]: Reação do professor / aprovação
[4,5s+]: Narração sobre o método
```

## GANCHO VISUAL 2: FRUTO PROIBIDO (BLUR / DESFOQUE)

**Descrição:** Imagem propositalmente desfocada ou pixelada.

**Psicologia:** O cérebro ODEIA informação incompleta. Quer completar o que está borrado.

**Como fazer:**
1. Imagem/vídeo começa borrado
2. Headline: "Desbloqueie este presente grátis"
3. Revela aos poucos OU manda clicar

**Headlines típicas:**
```
"Se você está vendo este vídeo secreto..."
"Desbloqueie esta informação proibida"
"[CENSURADO] — Clique para revelar"
```

**Quando usar:**
- Ofertas exclusivas
- Conteúdo "proibido"/conspiratório
- Nicho adulto (com cuidado)

**Adaptação King:**
```
[Tela borrada com resultado de aluno]
Headline: "O resultado dessa aluna em 6 meses vai te chocar"
[Revela: print de promoção/aumento salarial]
```

## GANCHO VISUAL 3: NARRAÇÃO POV (1a PESSOA)

**Descrição:** Vídeo filmado do ponto de vista dos seus olhos. Câmera mostra suas mãos enquanto narra.

**Como fazer:**
- Câmera mostra suas mãos
- Você manuseia produto/app
- Narra enquanto faz

**Por que funciona:** Imersão total — espectador sente que está fazendo.

**Exemplos genéricos:**
```
"Abrindo o app pela primeira vez..."
[Câmera mostra tela do celular + suas mãos]

"Preparando a receita que me fez perder 8kg..."
[Câmera mostra ingredientes + suas mãos cortando]
```

**Nichos ideais:**
- Unboxing
- Tutoriais
- Receitas
- Apps/software
- Artesanato

**Adaptação King:**
```
"Entrando na minha primeira aula ao vivo com professor particular..."
[Câmera mostra tela do notebook + mãos digitando]
[Professor aparece na tela, cumprimenta]
[Aluno começa a falar — câmera captura a interação]
```

```
"Testando a IA QUEEN pela primeira vez..."
[Câmera mostra celular + mãos tocando na tela]
[Digita: "Simular entrevista de emprego"]
[IA responde — narração reage]
```

## GANCHO VISUAL 4: GREEN SCREEN EFFECT

**Descrição:** Pessoa recortada sobre fundo de contexto (notícia, gráfico, print, outro vídeo).

**Como fazer:**
- Filtro nativo TikTok/Instagram OU
- Editor simples (CapCut, InShot)
- Fundo: Notícia, site, gráfico, outro vídeo

**Quando usar:**
- Gancho "Notícia" ("Atriz global implora...")
- Gancho "React" (reagindo a outro vídeo)
- Gancho "Dados" (gráfico ao fundo)
- Gancho "Fofoca" (print de rede social)

**Exemplo genérico:**
```
[Fundo]: Print de notícia "Inflação bate recorde"
[Pessoa na frente]: "E é por isso que você precisa de renda extra..."
```

**Adaptações King:**
```
[Fundo]: Manchete "78% das vagas de tech exigem inglês"
[Pessoa na frente]: "E você ainda tá esperando o momento certo pra começar..."

[Fundo]: Print do LinkedIn com vaga remota em dólar
[Pessoa na frente]: "Olha essa vaga. R$ 25 mil por mês. Requisito: inglês fluente."

[Fundo]: Gráfico "Salário médio com e sem inglês"
[Pessoa na frente]: "A diferença é de 52%. Você quer ficar de qual lado?"

[Fundo]: Print de depoimento de aluno King
[Pessoa na frente]: "Ela começou travada. Em 8 meses, tava liderando call internacional."
```

## GANCHO VISUAL 5: ASSALTO DO PRODUTO

**Descrição:** Abordagem "agressiva" positiva. Formato de entrevista de rua interrompendo pessoas.

**Estrutura:**
```
[Pessoa andando na rua]
[Você aparece]: "Espera, espera, espera!"
[Para a pessoa]: "Você PRECISA conhecer isso aqui"
[Apresenta produto]
```

**Por que funciona:**
- Quebra padrão de "anúncio de estúdio"
- Parece conteúdo orgânico
- Alto entretenimento

**Pode usar atores/amigos.**

**Adaptação King:**
```
[Pessoa no escritório/coworking]
"Espera! Quanto tempo você estuda inglês?"
"5 anos? E fala fluente?"
"Não? Deixa eu te mostrar uma coisa..."
[Mostra app/plataforma King]
```

```
[Pessoa na rua]
"Fala 'Good morning' pra mim"
[Pessoa fala com sotaque]
"Agora imagina ter um professor particular que corrige isso em tempo real, 4x por semana, por menos de R$ 300/mês"
```

## GANCHO VISUAL 6: LIFESTYLE / VLOG

**Descrição:** Intimidade do criador. Formato de conteúdo orgânico de influenciador.

**Formatos:**
```
"Veja como eu me arrumo" (GRWM — Get Ready With Me)
"Venha jantar comigo"
"Minha rotina de skin care"
"Um dia na minha vida [profissão]"
"Como atinjo o shape da minha vida"
```

**Por que funciona:**
- Não parece anúncio
- Parece conteúdo de influenciador
- Alta identificação

**Estrutura temporal:**
```
[0-3s]: Momento íntimo/rotina
[3-15s]: Insere produto naturalmente
[15s+]: Continua rotina + benefícios
```

**Adaptação King:**
```
"Um dia na minha vida como profissional bilíngue"
[0-3s]: Acordando, café, rotina matinal
[3-15s]: "Antes da reunião eu pratico 15 minutos na QUEEN"
[Mostra tela do celular com IA]
[15s+]: Entra na reunião, fala em inglês com confiança

"Como eu estudo inglês sem parar minha rotina"
[0-3s]: No carro, no trânsito
[3-15s]: Abre app, pratica com IA pelo áudio
[15s+]: Chega no trabalho, já praticou 15 minutos
```

## GANCHO VISUAL 7: DUAS TELAS (VÍDEO SATISFATÓRIO)

**Descrição:** Tela dividida — vídeo satisfatório em cima + pessoa falando/vendendo embaixo.

**Como fazer:**
1. Vídeo satisfatório (derretendo, cortando, ASMR)
2. Você falando/vendendo na parte de baixo
3. Narração sincronizada

**Por que funciona:**
> "Você tem duas formas de prender atenção simultâneas. Visual satisfatório + mensagem verbal."

**Exemplos de vídeo satisfatório:**
- Coisas derretendo
- Cortes perfeitos
- Sabão sendo cortado
- Slime
- Arte sendo feita

**Adaptação King:**
```
[Tela de cima]: Vídeo satisfatório de caligrafia/escrita
[Tela de baixo]: Pessoa falando sobre como destravar inglês

[Tela de cima]: Montagem rápida de before/after (prints de progresso)
[Tela de baixo]: Narração sobre o método King

[Tela de cima]: ASMR de digitação + tela do app
[Tela de baixo]: Aluno contando experiência
```

## FONTES DE PESQUISA PARA GANCHOS VISUAIS

### 1. TikTok — Ganchos Visuais Validados

**Método Hashtag:**
```
#[problema]
#emagrecimento
#aprenderingles
```

**Método Busca Alfabética:**
```
"você sabia" + [espaço] + A
"você sabia" + [espaço] + B
...
= Sugestões de buscas populares
```

**Método Análise de Virais:**
- Filtrar por "Mais relevantes"
- Ver vídeos com 1M+ views
- Anotar estrutura do gancho visual
- Adaptar para King

### 2. YouTube — Thumbnails como Ganchos

- Capas são ganchos visuais testados com investimento real
- Ver títulos + imagens de vídeos com milhões de views
- Identificar padrões visuais (cores, expressões faciais, composição)
- Adaptar para seu formato

### 3. Biblioteca de Anúncios (Facebook)

1. facebook.com/ads/library
2. Buscar concorrentes de escolas de inglês
3. Filtrar "Ativos"
4. Analisar:
   - Ganchos visuais repetidos (= estão funcionando)
   - Formatos dominantes
   - Primeiros frames do vídeo

### 4. Google Autocompletar

```
[tema] + [espaço]
"emagrecer " → Sugestões automáticas
"aprender inglês " → Sugestões automáticas

Seção "As pessoas também perguntam":
- Cada pergunta = gancho pronto
```

### 5. Reddit (Ouro Puro)

**Por quê:** Anonimato = verdade crua

**Como usar:**
1. reddit.com
2. Buscar em inglês: "learning english struggle", "english fluency"
3. Botão direito > Traduzir
4. Ler posts + comentários

**O que procurar:**
- Posts com 50+ comentários
- Desabafos emocionais
- Perguntas recorrentes
- Linguagem exata que usam

**Ouro extraído:**
- Dores específicas
- Personas escondidas
- Gatilhos emocionais
- Objeções reais

## ERROS FATAIS EM GANCHOS VISUAIS

### Erro 1: Ignorar o Componente Visual
- Focar só no roteiro (copy)
- Ignorar edição
- Não testar cortes rápidos/efeitos
- **Lembrete:** Gancho é VERBAL + VISUAL juntos

### Erro 2: Edição Estática
- Vídeo sem cortes nos primeiros 5 segundos
- Imagem sem contraste ou elemento de destaque
- Fundo genérico sem contexto

### Erro 3: Desalinhamento Copy x Visual
- Copy de urgência com visual calmo
- Copy de lifestyle com visual de estúdio corporativo
- Copy de dados com visual sem números/gráficos

### Erro 4: Não Testar Combinações
```
ERRADO: 1 copy + 1 visual = 1 anúncio
CERTO: 1 copy + 3 visuais = 3 anúncios para testar
```
